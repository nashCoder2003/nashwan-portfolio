# Personal Contact Webpage

A clean, responsive personal contact webpage built using HTML, CSS, and JavaScript. This project serves as a professional landing page where visitors can learn about me and get in touch easily.

## 🚀 Features

- Responsive layout for desktop and mobile
- Simple and professional UI
- Contact section for easy communication
- Lightweight and fast-loading
- Clean semantic HTML structure

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript  
- Git & GitHub Pages (for deployment)
